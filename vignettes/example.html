Datasets

Thereâ€™s three files needed to run the deltaGLM code. The package includes several that have been used as defaults for west coast assessments. It is critical that any new data used by the package have the same column names.

First, contains the area, in hectares of 1756 strata, defined by the latitude and longitude (in decimal degrees), depth (in meters). Each strata also has an associated area, in hectares. These individual areas are summed up to define areas for boundaries chosen by the user.

data(SA3)

Second, a data frame needs to be defined, with names (the northern boundary), (the southern boundary), (the minimum depth), and (the maximum depth). An example of generating this data frame is below. Third, a data frame with raw haul data for an individual species needs to be defined. We provide an example in the object.

data(Example_Species)
